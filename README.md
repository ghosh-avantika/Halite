# Halite Competition
All the files from the Halite Competition 2016-2017. Used Python in the Spyder environment to create a strategy for winning the Halite competition, which is an AI challenge where participants write bots to compete in an online multiplayers game.

## Files
Final Halite Submission
- This file contains the strategy my team used to compete in the Halite competition, including finding enemy targets, and strategic movement.

Movement Strategy
- This file contains the different types of movement strategies my team tested in order to avoid enemies in the game, while gaining the most points.
